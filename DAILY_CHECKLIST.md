# 📋 每日发布清单

## ⏰ 每日任务（3 篇 + 争议）

### 9:00 - 数据篇
- [ ] 主题：行业亏损/财报数据
- [ ] 要求：具体数字 + 来源 + **争议结论**
- [ ] 示例："AI 公司亏损上千亿，烧的都是投资人钱"

### 15:00 - 事件篇
- [ ] 主题：裁员/政策/新闻
- [ ] 要求：最新事件 + 媒体出处 + **民科点评**
- [ ] 示例："AI 裁员 8000 人，招的全是 Prompt 工程师？"

### 21:00 - 观点篇
- [ ] 主题：争议分析/深度思考
- [ ] 要求：极端观点 + 数据支撑 + **够"杠"**
- [ ] 示例："深度学习就是统计学披了层皮"

### 22:00 - 总结
- [ ] Git 提交今日内容
- [ ] 记录到 post_history.json
- [ ] 复盘今日数据

---

## 🔥 争议性检查

发布前确认：

- [ ] 有争议点（能引发反驳）
- [ ] 数据支撑（不是空喊）
- [ ] 说话够"杠"（不是和稀泥）
- [ ] 观点极端但有依据
- [ ] 能让读者"卧槽"或"傻X"

## 禁止项

- ❌ "AI 有优点也有缺点" - 和稀泥
- ❌ "AI 是把双刃剑" - 正确的废话
- ❌ "要看具体场景" - 没态度
- ❌ "因人而异" - 没观点
