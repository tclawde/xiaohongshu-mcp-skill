# ⚡ 快速参考

## 重要：不要暴露意图

### 不要暴露 ❌
- 民科
- 独立思考
- 反主流

### 标签自然 ✅
- 职场
- 就业
- 焦虑
- 流量

## 话题范围

### 可以谈 ✅
- 社会热点（就业、房价、教育）
- 生活话题（职场、人际关系、焦虑）
- 现象观察（网红、流量、炒作）
- 文化现象（躺平、内卷、焦虑）

## 内容要求

### 有厚度 ✅
- 近期热点新闻
- 具体案例
- 真实数据
- 普遍共鸣

### 不要只输出观点 ❌
- 有事件
- 有共鸣
- 有数据
- 自然表达

## 发布检查

- [ ] 标签没有暴露身份
- [ ] 话题是社会热点或共鸣
- [ ] 内容有事件/数据支撑
- [ ] 观点独特但不刻意
- [ ] 像普通人有感而发
